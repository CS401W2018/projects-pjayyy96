Why do we need a viewport meta tag?

The < meta > tag gives the browser instructions on how to control the page's dimensions and scaling. Additional, to be able to see the actual picture of your page on different platform.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The second or the lastly one in the CSS, in this case if :row come after column. The answer will be :row.

How do you change the CSS for a web page displayed on a phone?

By using the @media and put the max-width of the content and when it appears on the phone with the smaller size the content will decrease the size, and it will look better on it.
