What is the main difference between CSS Grid and CSS Flex?

CSS Grid provides two-dimensional layout capabilities, allowing for elements to be arranged horizontally and vertically.

CSS Flexbox provides flexibility, allowing for elements to be placed on a single axis, either horizontally or vertically.
