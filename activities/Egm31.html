Question 1: Explain the difference between inline, embedded, and external CSS.

Answer : Inline styles are those that are used as part of the HTML tag itself. An external style sheet is a separate text file that each page in the web site can link to it in order to receive it's instructions.

Question 2: Explain why you should only use external CSS.

Answer : To reduces the HTML file size and can be cached for faster page loads, which can affect the page load time.

Question 3: What is a selector used for?

Answer : It is a pattern of elements and other terms that tell the browser which HTML elements should be selected.
