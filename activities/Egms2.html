Question 2: What is the difference between an ID and a Class?

Answer: The ID name cannot be repeated within the HTML document. IDs are used to apply a specific style to an element, A class name can be used for multiple HTML elements. Classes are not unique, so the same class name can be used for multiple elements on a page.

Question 3: What is the difference between a SPAN tag and a DIV tag?

Answer: span tags are generally used for line-level content, such as short strings or individual words. div tags are generally used to group larger blocks, such as paragraphs or entire sections.

Question 5: What are the different ways to select a color?

Answer: It can use the style attribute to change the color of text or the background color of an HTML element.

  • Color name : for example, "Blue"
  • RGB : for example, "rgb(138,200,156)"
  • Hex : for example, "Pink #FFC0CB (a soft pink)"
  • HSL: For example, "hsl(340, 50%, 80%)"
  • Short code: for example, "#FFC0CB" F = FF, etc.
