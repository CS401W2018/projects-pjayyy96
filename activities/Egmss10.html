Questions!

When do you use a table?

We use table when we want to present data in an organized way that's easy to read, compare, and analyze. Tables are especially useful when you have a lot of data and Information, or when you want to highlight patterns or trends.

When do you not use a table?

We shouldn't use tables for layout purposes, as they can create accessibility issues and are not designed for that purpose. Instead, use CSS for layout and design, and use tables only to present tabular form.

When do you need to do extra work for table accessibility?

To make sure that we create the page for every user and provide a way to help them find their needs.
