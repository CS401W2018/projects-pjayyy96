Internet

Internet, sometimes simply called the net, is a worldwide system of interconnected computer networks and electronic devices that communicate with each other using an established set of protocols.

Web

A website is a collection of files accessed through a web address, covering a particular theme or subject, and managed by a particular person or organization. Its opening page is called a home page.

What is Git?

Git is a free, open-source version control system (VCS) that helps developers track and manage changes to their code. It's the most widely used VCS in the world and is considered the standard for software development.

What is GitHub

GitHub is a web-based platform that allows developers to store, share, and collaborate on code. It's a social coding platform that uses Git, an open source version control system, to track changes to code.
