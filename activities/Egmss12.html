Questions!

Why don't we use the border attribute?

The border attribute for HTML tables is deprecated in favor of CSS styling, and a way to control borders and other visual aspects of web pages. Without border, without line in the table.

What is the easiest way to color every other row a different color?

  • Select the table or an element in it.
  • Click Inspector in the toolbar and then click the Table inspector button.
  • Select Alternating Row Color.
  • Click the adjacent color well to open the Colors window, and then choose a color for the alternate rows.
