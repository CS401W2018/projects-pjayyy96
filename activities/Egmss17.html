The form tag has an "action" attribute. What does it do?

The action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.

The form tag has a "method" attribute. What does it do?

The method attribute in an HTML (form) tag specifies the HTTP method used to send form data to the server, either as URL variables (with method="get") or as an HTTP post transaction (with method="post")

What is the difference between a "POST" and a "GET"?

We use GET when you want to retrieve data and use POST when you want to submit data. GET is often used for simple data retrieval, like fetching a web page, while POST is used for more complex operations, such as submitting a form or uploading a file.
