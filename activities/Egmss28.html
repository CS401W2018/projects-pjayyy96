What is AJAX?

It is the short code name of JavaScript, AJAX is a set of web development techniques that allows web pages to update content without reloading the entire page.

Why do you put your JS Code in a separate file?

It makes it more readable and make it faster for loading. This separation makes it easier to manage and update your code.
