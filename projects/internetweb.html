Internet

Internet is a worldwide connection of computer networks and electronic devices that use to communicate with each other, such as, WWW stand for (World wide website), and other protocals. A platform that uses internet, for example, email, online banking, and streaming media.

Web

A website or world wide wed is a collection of files accessed through a web address that was originally developed at the European Laboratory. The web is including a particular theme or subject. Its opening page is called a home page that developed by person and organization.

These two types are introduce to the world wide network but internet is a global network of computers that are connected to eac other. The web is a service that runs on the internet. It's a content that can be accessed through browsers like Chrome, Firefox, or Microsoft edge.

ASCII and Unicode

ASCII is a type of character and it's limited to English alphabets and simple symbols. Moreover, ASCII is developed in 1960s by American National Standards Institute (ANSI). The main purpose is to standardize text encoding for English-speaking countries.

Unicode is a global standard language that supports most of the languges and symbol used in daily digital communication. Unicode is developed in 1991s by Unicode Consortium. The main purpose is to create a universal character encoding system to support all languges and symbols.

What is UTF-8

UTF-8 is the most popular character encoding on the website. It's also the way to encode Unicode characters. For example UTF-8 uses 1 to 4 bytes to show the character. Start with letter "A" is 01000001 (1 byte).

URL

URL stand for Uniform Resource Lacator, is a address of web on the internet. With that it tells your computer to find a specific webpage.

Domain Name

Domain name should be any names that human could read and it should be readable, for example, www.github.com, www.instagram.com. these are domain names.

HTTP

HTTP is the foundation of data exchange around the world, and it allows user to load web pages by just typing an URL. The recent version that published by QUIC protocal in 2020s, it called HTTP/3.

HTTPS

HTTPS is similar to HTTP but it added the secure data into the version, and HTTPS use encryption to protect the data, which it using the process of coverting data into a coded format to help protect from unauthorized access.

IP address

IP means Internet Protocal, and the role of IP is to allows any devices that connected to a internet communicate with each other. The format that uses is a numerial identifier, for example, IPv4 uses format xxx.xxx.x.x or 192.168.1.1

DNS

DNS is Domain names system, and it's a tool that works like a translater. Which it translates domain names to IP adress, the goal is to load web browsers correctly. This idea invented by Paul Mockapetris in 1983s.

W3C Recommendations

W3C is the standard worldwide web that includes the guidelines and practices for web technologies. This web reaches the "Recomendation" status, which considered secure, usable, and ready for all web works.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is the digital language uses to store, command, and passing data for both humans and machines.

XHTML (eXtensible Hypertext Markup Language) is used in older web development practices, to make sure all tags are properly closed and follow the XML standards.

HTML (Hypertext Markup Language) is the standard of creating web pages, and also defines content, layout, and aspect of a webpage.

HTML5 is the latest version of HTML. It performs a elements for modern webpage and application. HTML5 improves many new key features, such as the support for audio, video, plugins, and new term called APIs.
